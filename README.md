# Spam Message Data Analysis

## Project Overview
This project performs data analysis on a spam message dataset to identify the distribution of spam and ham messages using Python and data visualization techniques.

## Objective
To analyze message classification data and visualize the distribution of spam and non-spam (ham) messages.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Features
- Dataset loading and exploration
- Missing value analysis
- Spam vs Ham message analysis
- Data visualization using bar charts
- Exploratory Data Analysis (EDA)

## Dataset
Spam Message Dataset

## Workflow
1. Loaded the dataset using Pandas
2. Explored dataset structure and information
3. Checked missing values
4. Analyzed spam and ham message distribution
5. Created visualization for message classification

## Output
The project generates analysis tables and visualizations representing spam and ham message distributions.
